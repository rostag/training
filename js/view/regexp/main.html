Регулярные выражения - это паттерны, используемые для поиска совпадений комбинаций символов в строках. В JavaScript регулярные выражения также являются объектами.

Для поиска соответствий паттерну используются методы как объекта RegExp, так и объекта String.

Объект RegExp

Создает объект регулярного выражения для определения соответсвия текста паттерну.

Синтаксис:
/pattern/flags
new RegExp(pattern[, flags])

где, pattern - текст регулярного выражения (паттерн)
flags - опциональные параметры, используемые для настройки поиска в подстроке.

Пример испсользования:
var regexp = /pattern/;
var regexp = /pattern/gim;
var regexp = new RegExp('pattern');
var regexp = new RegExp('pattern', 'gim');

Паттерн
Шаблон (паттерн) регулярного выражения состоит из простых символов, таких как /abc/ или комбинации простых и специальных символов, таких как /ab*c/ или /Chapter (\d+)\.\d*/. Последний пример включает в себя скобки, которые используются в качестве запоминающего способа, т.е. найденное совпадение запоминается для последующего использования.

Флаги
Регулярные выражения могут иметь флаги (параметры), которые влияют на поиск:
i - Чувствительность к регистру. Если этот флаг есть, то регэксп ищет независимо от регистра, то есть не различает между А и а.
g - Глобальный поиск. Если этот флаг есть, то регэксп ищет все совпадения, иначе — только первое.
m - Многострочный поиск.

Создание регулярного выражения

Существует 2 способа создания объекта RegExp:

  1. С помощью литерала:

    var re = /ab+c/;
    Литерал предоставляет компиляцию регулярного выражения, когда скрипт загружается. Т.е. регулярное выражение всегда статично. Для определения не используются кавычки, также используется для лучшей производительности.

  2. С помощью функции-конструктора RegExp:

    var re = new RegExp("ab+c");
    Коструктор предоставляет компиляцию регулярного выражения во время выполнения. Используется когда регулярное выражение может меняться либо паттерн не известен и может быть получен из другого источника, такого как пользовательский ввод.
