Основным инструментом работы и динамических изменений на странице является DOM (Document Object Model). Согласно DOM-модели, документ является иерархией. Проще говоря, DOM - это представление документа в виде дерева тегов.

Подключение <script>

<script type="text/javascript">...your code...</script>
<script type="text/javascript" src="path/to/your.js"></script>

  • Подключение в секции HEAD
  • Подключение в секции BODY

Доступ к элементам

<html> и <body>

document.documentElement - Свойство ссылается на DOM-объект для тега HTML.
document.body - Свойство ссылается на DOM-объект для тега BODY. При доступе к document.body в момент обработки тега HEAD, то document.body = null, потому что BODY еще не существует.

Дочерние элементы

  • childNodes

    Псевдо-массив childNodes хранит все дочерние элементы, включая текстовые.

    var nodes = document.body.childNodes;
    for(var i = 0, l = nodes.length; i < l; i++){
      console.log(nodes[i]);
    }

  • children

    Свойство children, перечисляет только дочерние узлы, соответствующие тегам

  • firstChild и lastChild

    Свойства обеспечивают быстрый доступ к первому и последнему потомку.

Parent and Sibling

  • parentNode

    Свойство ссылается на родительский узел.

  • previousSibling, nextSibling

    Свойства дают доступ к левому и правому соседу

Методы для создания элементов

  • document.createElement(tag)

    Создает новый элемент с указанным тегом

    var div = document.createElement('div');

  • elem.cloneNode(boolean)

    Клонирует элемент elem, вместе с атрибутами, true - включая вложенные в него, false - без подэлементов

    var newElem = elem.cloneNode(true);

Вставка и удаление элементов

  • parent.appendChild(elem)

    Добавляет elem в список дочерних элементов parent. Новый элемент добавляется в конец списка

    document.body.appendChild(newElem);

  • parent.insertBefore(elem, nextSibling)

    Вставляет elem в список дочерних parent, перед элементом nextSibling.

  • parent.removeChild(elem)

    Удаляет elem из списка дочерних элементов parent.

  • parent.replaceChild(elem, currentElem)

    Заменяет currentElem на elem.

Содержимое элемента

  • innerHTML

    Позволяет получить HTML-содержимое элемента в виде строки. В innerHTML можно и читать и писать.

  • outerHTML

    Свойство outerHTML содержит HTML элемента целиком. Можно и записать outerHTML, при этом новый текст будет вставлен вместо элемента.

Свойства и атрибуты

Свойства

Узел DOM - это объект, поэтому, как и любой объект в JavaScript, он может содержать пользовательские свойства и методы.

document.body.someData = '123';
document.body.someMethod = function(){};

Атрибуты

  • elem.hasAttribute(name) - проверяет наличие атрибута
  • elem.getAttribute(name) - получает значение атрибута
  • elem.setAttribute(name, value) - устанавливает атрибут
  • elem.removeAttribute(name) - удаляет атрибут

Стандартные свойства DOM синхронизируются с атрибутами.

Браузер синхронизирует стандартные атрибуты со свойствами.

elemtent.id = 'someId'; // div id="someId"
a.href = 'www.somesite.com' // a href="www.somesite.com"
input.value = 'new value'; // input type="..." value="new value"

Атрибут "class" соответствует свойству className

Поиск элементов

  • getElementById

    Поиск по атрибуту Id

    document.getElementById(id);

  • getElementsByName

    Все элементы с заданным тегом tag внутри элемента elem и возвращает их в виде списка

    document.getElementsByTagName('div');

    document.getElementsByTagName('*'); - найдет все элементы в документе

  • getElementsByTagName

    Все элементы с данным атрибутом name.

  • getElementsByClassName

    Возвращает коллекцию элементов с классом className. Находит элемент и в том случае, если у него несколько классов, а искомый - один из них. IE9+

  • querySelectorAll

    Возвращает все элементы внутри elem, удовлетворяющие CSS-селектору cssQuery. IE9+

    var elements = document.querySelectorAll('ul > li:last-child');

  • querySelector

    Возвращает 1-й элемент внутри elem, удовлетворяющие CSS-селектору cssQuery.
