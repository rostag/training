Створення об'єктів

Об'єкти створюються шляхом виклику спеціальних функцій-конструкторів:

 var o = new Object();
 var now = new Date();
 var pattern = new RegExp("\\sjava\\s", "i");

Створивши власний об'єкт, можна його як завгодно використовувати і встановлювати його властивості

 var point = new Object();
 point.x = 2.3;
 point.y = -1.2;

JavaScript визначається синтаксис об'єктних літералів, що дозволяє створювати об'єкти і вказувати їх властивості. Об'єктний літерал (також відомий як Ініціалізатор об'єкта) є списком розділених комами пар «властивість / значення», укладений у фігурні дужки. Усередині пар роль роздільника грає двокрапка. Таким чином, об'єкт point з попереднього прикладу також може бути створений і ініційований наступним рядком:

 var point = { x:2.3, y:-1.2 };

Об'єктні літерали можуть бути вкладеними.

 var rectangle = { upperLeft: { x: 2, y: 2 } };

Значеннями властивостей в об'єктних літералах не обов'язково повинні бути константи - це можуть бути довільні JavaScript-вирази.

 var square = { "upperLeft": { x:point.x, y:point.y } };
