Шаблони створення об'єктів

Створення об'єктів в JavaScript виконується досить просто - для цього достатньо скористатися літералом об'єкта або викликати функцію-конструктор.

Ці шаблони допоможуть вам організувати і структурувати програмний код в додатках і послабити вплив глобальних змінних.

Далі розглянуті кілька прийомів, використовуваних при створенні об'єктів.
