JavaScript позволяет работать с тремя элементарными типами данных: числами, строками и логическими значениями.

В JavaScript также определяются два тривиальных типа данных, null и undefined, каждый из которых определяет только одно значение.

В дополнение к этим элементарным типам данных JavaScript поддерживает составной тип данных, известный как объект.

В JavaScript определен еще один специальный тип объекта, известный как функция (function).
