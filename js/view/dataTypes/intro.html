Вітаємо! Тут ви знайдете декілька базових уроків з Javasript.

Програма

Цей курс складається з тем, уроків, домашніх завдань та відеозаписів. Щоб почати роботу, виберіть тему для вивчення у меню (зліва), або завантажте один з файлів уроків (див. нижче)

Уроки і домашня робота

Ось — файли уроків курсу JavaScript для початківців. У кінці кожного файлу ви знайдете домашнє завдання. Опануйте урок і виконайте домашню роботу перед тим, як перейти до наступного:

  1. Файл l-01.js. Теми: Мова і типи даних JavaScript.
  2. Файл l-02.js. Теми: Прості типи даних.
  3. Файл l-03.js. Теми: Спеціальні та складні типи даних, області видимости.
  4. Файл l-04.js. Теми: Оголошення змінних та функцій, глобальне та локальне, ланцюги видимости.
  5. Файл l-05.js. Теми: Вирази і оператори.
  6. Файл l-06.js. Теми: Оператори, оптимізація коду, поняття рефакторингу.
  7. Файл l-07.js. Теми: Інструкції, базові методи масивів.
  8. Файл l-08.js. Теми: Просунуті методи масивів.
  9. Файл l-09.js. Теми: Об'єкти: створення, властивості, методи, JSON, DOM та BOM
  10. Файл l-10.js. Теми: Функції: види визначень, параметри та області видимости, стек викликів, замикання, побічні ефекти.
  11. Файл l-11.js. Теми: Приклади замикань, анонімні функції, рекурсія.
  12. Файл l-12.js. Теми: Деякі шаблони JavaScript.
  13. Файл l-13.js. Теми: Регулярні вирази.
  14. Файл l-14.js. Теми: Практичне використання колбеків і замикань на прикладі простого застосунку.

Література

Є багато гарних книжок з Javascript. Наведемо улюблену:

Eloquent Javascript

Ця сучасна книга написана яскравою мовою, має багато цікавих прикладів, є безкоштовна PDF-версія

Відеозаписи

Також ви можете передивитися відеозаписи (увага: доступ може бути обмеженим).

Known Issues

  1. Файли уроків покривають не усі теми, а переважно базові. Для глибшого вивчення варто пройтися по усих темах у меню зліва, навіть якщо ви пройшли усі уроки.
  2. Цей курс доповнюється новими матеріалами.
  3. В файлах уроків згадуєьться Shell. Де його взяти, ви можете дізнатися у свого менеджера. Але це не обов'язково для проходження курсу.
  4. Деякі уроки них ще чекають перекладу з англійської чи російської мови (ви можете допомогти з цим).
