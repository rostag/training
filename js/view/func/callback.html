Функції зворотного виклику (Callback pattern)

Функції - це об'єкти, тобто вони можуть передаватися як аргументи інших функцій. Якщо, наприклад, функція bar() передається як параметр функції foo(), це оначає, що в якийсь момент функція foo() виконає (або викличе) функцію bar(). У цьому випадку функція bar() називається функцією зворотного виклику:

function foo(callback){
  // some code
  if(callback typeof === 'function') callback()
  // some code
}

function bar(){
  // some code
}

foo(bar);

Зверніть увагу, що при передачі функції у вигляді аргументу вона передається без круглих дужок. Наявність круглих дужок викликає виконання функції, тоді як в даному випадку нам необхідно всього лише передати посилання на цю функцію і дати функції foo () можливість виконати її (тобто провести про- ратний виклик) в потрібний момент часу.
