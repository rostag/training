Кожна функція - це замикання

Тіло функції виконуєься в локальній області видимості, яка відрізняється від глобальної.

Функції в JavaScript є комбінацією виконуваного коду і області видимості, в якій цей код виконується. Така комбінація програмного коду і області видимості називається замиканням (closure). замиканням (closure).

Всі функції є замиканнями. Тобто функції "замикають" на собі змінні всіх зовнішніх функцій або мають доступ до глобальної / зовнішнім областям видимості - Scope chain.

Однак всі ці замикання представляють інтерес тоді, коли вкладена функція експортується за межі області видимості, в якій вона була визначена — тобто, коли вкладена функція передається у вигляді значення, що повертається зовнішній функції або зберігається у вигляді властивості будь-якого іншого об'єкта.

У такому випадку з'являється зовнішнє посилання на вкладену функцію, при цьому вкладена функція продовжує посилатися на простір імен зовнішньої функції. В результаті всі простори імен (об'єкти виклику), створені при кожному такому зверненні до зовнішньої функції, продовжують своє існування, а разом з ними продовжують існування імена і значення аргументів функції і локальних змінних.

function counter(){
  var count = 0;
  return function(){
    console.log(++count);
  }
}
var increment = counter();
increment(); // 1
increment(); // 2
increment(); // etc.
