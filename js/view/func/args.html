Функції в JavaScript можуть викликатися з довільним числом аргументів незалежно від того, скільки аргументів зазначено у визначенні іменованої функції. Оскільки функції типізовані слабо, то відсутня можливість задавати типи вхідних аргументів, в зв'язку з чим вважається допустимим передавати значення будь-яких типів будь-яких функцій.

Коли функція викликається з меншою кількістю аргументів, ніж описується в визначенні, відсутні аргументи отримують значення undefined.

У тілі функції ідентифікатор arguments завжди має особливий сенс; arguments - це спеціальна властивість об'єкта виклику, що посилається на об'єкт, відомий як об'єкт Arguments - це щось на зразок масиву, який дозволяє витягувати передані функції значення за номером, а не по імені.

foo(1,2,3)
function foo(){ console.log(arguments); }

foo(1,2,3)
function foo(a,b){ console.log(arguments.length); }
