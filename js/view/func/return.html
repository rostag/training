Повернення функцій

Функції - це об'єкти, тому їх можна використовувати в якості значень, що повертаються. Тобто функція не обов'язково повинна віз обертати якесь конкретне значення або масив даних. Функція може повертати іншу, більш спеціалізовану функцію або створювати іншу функцію в міру необхідності в залежності від вхідних параметрів.

var setup = function () {
  alert(1);
  return function () {
    alert(2);
  };
};

var my = setup(); //alert 1
my(); // alert 2

Оскільки функція setup() обгортає функцію, яку повертає, то утворюється замикання, яке можна використовувати для зберігання деяких приватних даних, доступних функції, що повертається, і недоступних за її межами. Прикладом таких приватних даних може служити лічильник у замиканні.
